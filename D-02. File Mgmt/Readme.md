2️⃣ File Management (Command, Description & Example)


touch:
Create an empty file
touch index.html


cp:
Copy files or directories
cp file1.txt /home/user/


mv:	
Move or rename files
mv file.txt /tmp/ or mv old.txt new.txt


rm:
Delete files	
rm file.txt


rm -r:	
Delete directory and its contents recursively	
rm -r myfolder/


cat:
Display file content
cat /etc/hostname


less / more:
View file content page by page	
less /var/log/syslog


head / tail:
Show first/last lines of a file	
head -n 10 file.log / tail -n 20 file.log


nano / vi:
Edit a file from terminal	
nano config.txt or vi config.txt


file:
Check file type	
file myscript.sh
