8️⃣ Package Management (Command, Description & Example)

apt update:
Update package lists (Debian/Ubuntu)
sudo apt update


apt install:
Install package
sudo apt install nginx


apt remove:
Uninstall package
sudo apt remove nginx


yum install:
Install package (CentOS/RedHat)
sudo yum install httpd


dnf update:
Update packages (Fedora/CentOS 8+)	
sudo dnf update


snap install:
Install snap packages
sudo snap install docker
