Introduction

In this module, I learned and practiced some *System Information and Process Management* Linux commands.

5️⃣ System Information & Process Management (Command, Description & Example)


uname -a:
Show system information
uname -a


hostname:
Display system hostname	
hostname


uptime:
Show how long system has been running	
uptime


top:
Show active processes and system usage
top


htop:
Advanced process viewer (if installed)	
htop


ps:
Display active processes	
ps aux


kill:
Kill a process by PID	
kill 1234


df -h:	
Check disk space usage	
df -h


du -sh:
Check directory size	
du -sh /var/log


free -h:
Show memory usage
free -h


whoami:
Show current logged-in user	
whoami


id:
Show user ID and group info
id
