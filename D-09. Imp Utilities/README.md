In this module, I learned and practiced *Important Utilities* Linux commands.

9️⃣ Important Utilities (Command, Description & Example)

date:
Show current date/time
date


cal:
Display calendar
cal


history:
Show previous commands
history


clear:
Clear terminal screen
clear


alias:
Create command shortcut
alias ll='ls -la'


echo:
Display text or variable
echo $HOME


man:
Open command manual
man ls
